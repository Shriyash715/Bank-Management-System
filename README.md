# 🏦 Bank Management System

A Java-based Bank Management System developed using Core Java, JDBC, and MySQL. This project demonstrates object-oriented programming concepts and database connectivity through a simple banking application.

## 🚀 Features

- Create New Account
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Transaction History
- MySQL Database Integration using JDBC

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- Eclipse IDE

## 📂 Project Structure

- `src/` – Java source code
- `Account.java`
- `BankApplication.java`
- `TestDB.java`

## 💾 Database

- MySQL
- JDBC Connectivity
- Account Information Storage
- Transaction Records

## ▶️ How to Run

1. Open the project in Eclipse.
2. Configure the MySQL database.
3. Update database credentials in the JDBC connection.
4. Run `BankApplication.java`.
